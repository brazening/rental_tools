# Аренда инструментов (Rental)

Программа для автоматизации работы пункта проката инструментов. Разработана на **C# WPF** с использованием **PostgreSQL** в качестве базы данных.

---

## 📋 Описание

Приложение позволяет вести учёт:

- Клиентов, сотрудников, поставщиков и складов;
- Инструментов (моделей, категорий, экземпляров);
- Договоров аренды, возвратов, ремонтов, списаний и перемещений;
- Формировать аналитические отчёты.

---

## 🚀 Основные возможности

### 📝 Раздел «Ввод»
- **Поставщики** – добавление информации о поставщиках.
- **Сотрудники** – ведение списка сотрудников компании.
- **Клиенты** – регистрация клиентов, берущих инструменты в аренду.
- **Категории** – создание классификатора инструментов (электроинструмент, ручной и т.д.).
- **Модели инструментов** – добавление моделей с указанием цены, залога и штрафа.
- **Города доставки** – перечень городов, куда возможна доставка.
- **Склады** – добавление и настройка складов хранения.

### 🔧 Раздел «Арендный парк»
- **Поставка** – оформление поступления инструментов от поставщиков.
- **Договор аренды** – заключение договора на аренду конкретного экземпляра.
- **Возврат** – оформление возврата, расчёт стоимости и штрафов.
- **Списание** – удаление из базы сломанных или утерянных инструментов.
- **Ремонт** – фиксация отправки в ремонт и возврата из ремонта.
- **Перемещение** – перемещение инструментов между складами.

### 📊 Раздел «Отчёты»
- **Рейтинг моделей** – анализ популярности моделей.
- **Продажи сотрудников** – эффективность работы сотрудников.
- **Поставки за период** – поступления инструментов за выбранный интервал.
- **Штрафы за период** – сводка по полученным штрафам.
- **История аренд по инструменту** – все аренды для конкретного экземпляра.

## 🛠 Технологии

- **Язык:** C# (.NET 6.0 / .NET Core)
- **UI:** Windows Presentation Foundation (WPF)
- **СУБД:** PostgreSQL (версия 12+)
- **Доступ к данным:** Entity Framework Core (или Dapper – уточняется в коде)
- **Стиль:** Современный тёмный интерфейс с удобной навигацией

---

## ⚙️ Требования

- Windows 7/10/11
- [.NET 6.0 Runtime](https://dotnet.microsoft.com/download) (или выше)
- PostgreSQL (локально или на сервере)
- Права на создание базы данных (для первого запуска)
